    ARGUMENTOS

    Quando executamos uma classe que contenha o método main, o mesmo permite que passemos
um array [] de argumentos do tipo "String". Logo, podemos, após a definição da classe a
ser executada, informar estes parâmetros da seguinte forma:
"java MinhaClasse argumentoUm argumentoDois" (sem aspas).

    Exemplo:
    public class AboutMe {
        public static void main(String [] args) {
            // Os argumentos começam com indice 0
            String nome = args[0];
            String sobrenome = args[1];
            int idade = Integer.valueOf(args[2])
            double altura = Double.valueOf(args[3])

            System.out.println("Olá!!! Me chamo " + nome + " " + sobrenome);
            System.out.println("Tenho " + idade + " anos");
            System.out.println("Minha altura é  " + altura + " m");
        }
    }
